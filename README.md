## What is ZuriPoll ?

ZuriPoll is a decentralized voting application that allows Zuri as an organization to set up elections and vote using a custom token

* Contract address: "0x18E30654BE206b778a5ad7Dd08278BcED20FbD11"

* Network: rinkeby
